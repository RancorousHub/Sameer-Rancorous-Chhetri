# Rancorous Hub Projects

Three ready-to-use web projects — no build tools or servers required.

## 1. SWizard - Sound Frequency Generator (`/frequency-generator`)
A precision sound frequency generator built with the Web Audio API. Features:
- Generate tones from 20 Hz to 20 kHz with real-time waveform and spectrum visualizer
- 4 waveform types: Sine, Square, Sawtooth, Triangle
- Preset libraries: Musical Notes, Tuning References, Scientific Frequencies, Binaural Beats
- Binaural beat mode (use headphones) for brainwave entrainment
- Frequency sweep with adjustable range and duration
- Fine-tune controls (+/- 0.1, 1, 10 Hz) and logarithmic slider
- Keyboard shortcut: Spacebar to play/stop

## 2. Freelance Site (`/freelance-site`)
Professional service business website for **Rancorous Hub**. Includes services, portfolio, testimonials, pricing, and contact form.

## 3. Affiliate Review Site (`/affiliate-site`)
Tech gadget review and affiliate site called **GadgetPick**. Includes product reviews, comparisons, deals section, and newsletter signup.

## How to Run

All three projects are static HTML/CSS/JS — no install, no build step, no server needed.

### Option A: Open locally
1. Clone or download this repository
2. Open any project's `index.html` directly in your browser:
   - `frequency-generator/index.html` — SWizard frequency generator
   - `freelance-site/index.html` — Freelance portfolio
   - `affiliate-site/index.html` — GadgetPick affiliate site

### Option B: GitHub Pages (live on the web)
1. Go to your repo **Settings > Pages**
2. Under "Build and deployment", set Source to **GitHub Actions**
3. Push to `main` — the included workflow will auto-deploy
4. Your sites will be live at:
   - `https://<username>.github.io/Sameer-Rancorous-Chhetri/` — Project hub
   - `https://<username>.github.io/Sameer-Rancorous-Chhetri/frequency-generator/` — SWizard
   - `https://<username>.github.io/Sameer-Rancorous-Chhetri/freelance-site/` — Freelance site
   - `https://<username>.github.io/Sameer-Rancorous-Chhetri/affiliate-site/` — GadgetPick
